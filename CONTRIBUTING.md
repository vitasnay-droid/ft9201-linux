# Contributing

Thanks for helping with FT9201 Linux support.

## Please do
- report exact laptop model
- report distro and kernel version
- report lsusb output for the fingerprint device
- describe what works and what fails
- share logs and protocol observations
- keep biometric privacy in mind

## Please do NOT upload publicly
- raw fingerprint images
- raw USB captures containing biometric payload
- personal biometric data

## Helpful contributions
- testing on additional hardware
- protocol decoding
- libfprint integration work
- code cleanup
- documentation improvements
